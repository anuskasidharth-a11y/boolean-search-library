# Boolean Search Basics

## What is Boolean Search?

Boolean Search is a search technique that combines keywords using operators to find more accurate candidate profiles.

Recruiters use Boolean Search to source candidates across LinkedIn, Google, GitHub, job boards, and resume databases.

---

# Main Operators

## AND

Narrows the search.

Example

```text
Java AND Spring
```

Returns profiles containing both Java and Spring.

---

## OR

Broadens the search.

Example

```text
Java OR Python
```

Returns profiles containing either Java or Python.

---

## NOT

Excludes unwanted results.

Example

```text
Java NOT Intern
```

Removes intern profiles.

---

## ""

Searches for an exact phrase.

Example

```text
"Software Engineer"
```

---

## ()

Groups similar terms.

Example

```text
("Software Engineer" OR Developer)
AND Java
```

---

# Example Search

```text
("Java Developer" OR "Software Engineer")
AND
(Java OR Spring)
NOT
Intern
```

---

# Common Mistakes

- Forgetting quotation marks.
- Using too many AND operators.
- Excluding too many keywords.
- Searching only one job title.
