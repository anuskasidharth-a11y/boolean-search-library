# Google X-Ray Search Guide

## What is Google X-Ray Search?

Google X-Ray Search is a sourcing technique used by recruiters to find candidate profiles that may not appear in normal LinkedIn searches. It uses Google's search engine with advanced search operators.

---

# Basic Syntax

```text
site:linkedin.com/in
```

This tells Google to search only LinkedIn profile pages.

---

# Example 1

```text
site:linkedin.com/in "Java Developer" Bangalore
```

### Purpose

Find LinkedIn profiles of Java Developers located in Bangalore.

---

# Example 2

```text
site:linkedin.com/in ("Java Developer" OR "Software Engineer") Bangalore
```

### Purpose

Search for candidates with either job title.

---

# Example 3

```text
site:linkedin.com/in ("Talent Acquisition Specialist" OR Recruiter) Bangalore
```

### Purpose

Find recruiters working in Bangalore.

---

# Example 4

```text
site:linkedin.com/in ("Python Developer") Hyderabad
```

### Purpose

Find Python Developers located in Hyderabad.

---

# Example 5

```text
site:linkedin.com/in ("Data Analyst") SQL Python
```

### Purpose

Find Data Analysts with SQL and Python skills.

---

# Common Google Operators

## site:

Limits results to one website.

Example:

```text
site:linkedin.com/in
```

---

## OR

Searches for multiple alternatives.

Example:

```text
(Java OR Python)
```

---

## ""

Searches for an exact phrase.

Example:

```text
"Software Engineer"
```

---

## -

Excludes a word.

Example:

```text
site:linkedin.com/in "Java Developer" -intern
```

---

# Best Practices

- Always use quotation marks for job titles.
- Use OR for alternate titles.
- Add skills after the title.
- Add city names for location-specific hiring.
- Keep searches simple before adding many filters.
