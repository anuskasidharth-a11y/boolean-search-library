# QA Engineer Boolean Search

## Manual Testing
```text
("QA Engineer" OR "Test Engineer")
AND
(Manual Testing OR Test Cases)
NOT
(Intern OR Trainee)
```

## Automation Testing
```text
("Automation Tester" OR "SDET")
AND
(Selenium OR Java OR Python)
```

## Performance Testing
```text
("Performance Tester")
AND
(JMeter OR LoadRunner)
```

## Purpose
Find QA professionals across manual, automation, and performance testing.

## Recruiter Tips
- Selenium + Java is most common
- Always separate manual and automation roles
- Add SDET for modern QA hiring
