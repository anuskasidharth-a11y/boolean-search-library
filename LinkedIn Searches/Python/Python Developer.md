# Python Developer Boolean Search

## Search String

```text
("Python Developer" OR "Python Engineer")
AND
(Django OR Flask OR FastAPI)
NOT
(Intern OR Trainee)
```

## Purpose

Find Python Developers with experience in Django, Flask, or FastAPI while excluding interns and trainees.

## Best For

- Backend Development
- API Development
- Product Companies
- Service Based Companies

## Platforms

- LinkedIn
- Google X-Ray

## How to Customize

For Bengaluru:

```text
AND Bengaluru
```

For AWS experience:

```text
AND AWS
```

For senior candidates:

```text
("Senior Python Developer" OR "Lead Python Developer")
```

## Recruiter Tips

- Search both Python Developer and Python Engineer.
- Add cloud skills when sourcing backend engineers.
- Include frameworks based on the job description.
