# Django Developer Boolean Search

## Search Strings

### Core Search
```text
("Django Developer" OR "Python Developer" OR "Backend Developer")
AND
(Django OR DRF OR "Django REST Framework")
NOT
(Intern OR Trainee)
```

---

### Advanced Search
```text
("Backend Engineer" OR "Python Developer")
AND
(Django AND (REST OR APIs))
AND
(PostgreSQL OR MySQL)
```

---

## Purpose

Find Python developers specialized in Django framework and backend development.

---

## Best For

- Backend Web Development
- API Development
- Product Based Companies
- SaaS Applications

---

## Recruiter Tips

- Always include DRF (Django REST Framework).
- Add database skills like PostgreSQL or MySQL.
- Use "Backend Engineer" for broader reach.
