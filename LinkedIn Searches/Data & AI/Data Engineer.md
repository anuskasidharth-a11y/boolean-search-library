# Data Engineer Boolean Search

## Core Search
```text
("Data Engineer")
AND
(Python OR SQL OR Spark OR Hadoop OR ETL)
NOT
(Intern OR Trainee)
```

## Data Analyst Search
```text
("Data Analyst")
AND
(SQL OR Excel OR Power BI OR Tableau)
```

## Data Scientist Search
```text
("Data Scientist")
AND
(Python OR Machine Learning OR AI OR TensorFlow)
```

## Purpose
Find professionals in analytics, data engineering, and AI roles.

## Recruiter Tips
- SQL is mandatory for analytics roles
- Python is key for AI roles
- Separate Analyst vs Engineer clearly
