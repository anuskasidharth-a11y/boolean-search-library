# .NET Developer Boolean Search

## Search Strings

### Core Search
```text
(".NET Developer" OR "C# Developer" OR "ASP.NET Developer")
AND
(.NET OR C# OR ASP.NET)
NOT
(Intern OR Trainee)
```

---

### Advanced Search
```text
("Senior .NET Developer" OR "Backend Engineer")
AND
(.NET AND (Microservices OR API))
AND
(Azure OR SQL)
```

---

## Purpose

Find .NET developers working on enterprise backend systems and web applications.

---

## Best For

- Enterprise Applications
- Backend Development
- Banking and Finance Systems
- Service Based Companies

---

## Recruiter Tips

- Always include both ".NET" and "C#".
- Add Azure for cloud-based roles.
- Include SQL for enterprise backend hiring.
