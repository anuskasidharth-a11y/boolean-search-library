# Spring Boot Developer Boolean Search

## Search Strings

### Core Search
```text
("Spring Boot Developer" OR "Java Developer" OR "Backend Developer")
AND
("Spring Boot" OR Microservices OR REST API)
NOT
(Intern OR Trainee)
```

---

### Microservices Focus
```text
("Backend Engineer" OR "Java Developer")
AND
(Spring Boot AND Microservices)
AND
(Docker OR Kubernetes OR AWS)
```

---

## Purpose

Find Java developers with strong Spring Boot and microservices experience for backend development roles.

---

## Best For

- Backend Development
- Microservices Architecture Roles
- Product Based Companies
- Scalable System Development

---

## Recruiter Tips

- Always include "REST API" for backend screening.
- Add Docker/Kubernetes for modern cloud-native roles.
- Use "Backend Developer" as a fallback title.
