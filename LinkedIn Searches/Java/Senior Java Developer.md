# Senior Java Developer Boolean Search

## Search Strings

### Core Search
```text
("Senior Java Developer" OR "Java Developer" OR "Backend Engineer")
AND
(Java OR Spring OR Spring Boot OR Microservices)
NOT
(Intern OR Trainee)
```

### Advanced Search
```text
("Senior Software Engineer")
AND
(Java AND (Spring OR Spring Boot))
AND
(AWS OR Azure OR Microservices)
```

## Purpose

Find experienced Java developers with strong backend and system design experience.

## Best For

- Senior Backend Roles
- Product Based Companies
- System Design Heavy Roles

## Recruiter Tips

- Always include "Backend Engineer" as alternate title.
- Add Microservices for modern Java roles.
- Include cloud (AWS/Azure) for senior candidates.
