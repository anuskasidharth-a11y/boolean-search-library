# Java Developer Boolean Search

## Search String

```text
("Java Developer" OR "Java Engineer" OR "Backend Developer")
AND
(Java OR Spring OR "Spring Boot")
NOT
(Intern OR Trainee)
```

## Purpose

This Boolean search helps recruiters find Java Developers with experience in Java and Spring technologies while excluding interns and trainees.

## Best For

- Technical Hiring
- Backend Development Roles
- Mid-Level Java Developers
- Product and Service Based Companies

## Platforms

- LinkedIn
- Google X-Ray

## How to Customize

To search in a specific city, add:

```text
AND Bangalore
```

To search for senior candidates, replace:

```text
"Java Developer"
```

with

```text
("Senior Java Developer" OR "Lead Java Developer")
```

To search for candidates with cloud skills, add:

```text
AND (AWS OR Azure OR GCP)
```

## Tips

- Use quotation marks for exact job titles.
- Use OR to include alternative titles.
- Use NOT to remove irrelevant profiles.
- Keep the search simple before adding many filters.
