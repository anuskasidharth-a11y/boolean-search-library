# Cloud Engineer Boolean Search

## AWS Focus
```text
("AWS Engineer" OR "Cloud Engineer" OR "DevOps Engineer")
AND
(AWS OR EC2 OR S3 OR Lambda)
NOT
(Intern OR Trainee)
```

## DevOps Focus
```text
("DevOps Engineer")
AND
(Docker OR Kubernetes OR Terraform OR Jenkins)
```

## Purpose
Find cloud and DevOps engineers with infrastructure experience.

## Recruiter Tips
- Always include Kubernetes
- Add Terraform for infra roles
- Include Jenkins for CI/CD
