# .NET Developer Boolean Search

## Core Search
```text
(".NET Developer" OR "C# Developer" OR "ASP.NET Developer")
AND
(.NET OR C# OR ASP.NET)
NOT
(Intern OR Trainee)
```

## Advanced Search
```text
("Senior .NET Developer" OR "Backend Engineer")
AND
(.NET AND (Microservices OR API))
AND
(Azure OR SQL)
```

## Purpose
Find .NET developers working on enterprise backend systems.

## Recruiter Tips
- Use both .NET and C# keywords
- Add Azure for cloud roles
- Add SQL for enterprise hiring
